About This Project

Yadav Authentication Project is a fully functional authentication demo built with HTML, CSS, JavaScript, and Firebase. It demonstrates all essential auth flows: sign up, login, Google sign‑in, password reset, email verification, and session management. This project is designed for my portfolio, showcasing my ability to integrate Firebase into a modern, responsive, and accessible frontend.

The code is clean, well‑organised, and follows the same design system as my other works — ensuring a consistent and professional appearance.
