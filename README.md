# AuthDashboard

A modern, responsive authentication dashboard template built with HTML, CSS, vanilla JavaScript, and Firebase. Ready to deploy on GitHub Pages.

## Features

- Email/Password sign-up and login
- Google Sign-In popup
- Password reset flow
- Protected dashboard and profile pages
- Route guards with redirect after login
- Universal notification system (no browser alerts)
- Responsive design with mobile navigation
- Modular CSS and JavaScript architecture
